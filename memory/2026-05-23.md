
## Timesheet Dashboard Filename Preference
- Pak Bos meminta: setiap kirim dashboard HTML timesheet, nama file harus menyertakan nama bulan dan tahun dalam Bahasa Indonesia.
- Format wajib: `Dashboard Timesheet <Bulan> <Tahun>.html`
- Contoh: jika diminta bulan Mei 2026, filename attachment: `Dashboard Timesheet Mei 2026.html`.

## Automation: Timesheet Report H-5 Akhir Bulan
- Pak Bos meminta schedule automation report timesheet: setiap H-5 akhir bulan, laporkan report timesheet di Telegram group Agent AI Command Center.
- Cron job dibuat: `Timesheet report H-5 akhir bulan ke Agent AI Command Center` (id: f79a3e71-779e-47dc-8937-efdf40c61e8e).
- Schedule cron: `0 9 23-26 * *` timezone `Asia/Jakarta`; agent mengecek apakah hari itu tepat H-5 dari hari terakhir bulan. Jika ya, generate/kirim dashboard timesheet bulan berjalan.
- Attachment filename wajib tetap mengikuti preferensi: `Dashboard Timesheet <Bulan> <Tahun>.html`.
